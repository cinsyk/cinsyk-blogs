## 良好习惯
``` javascript
// 短注释
/**
* 长注释
* @<name> 可以添加各种说明
*/
```
* 养成良好的注释书写习惯，方便快速阅读代码并修改

``` javascript
// 可变值：使用小写或驼峰式标识符
var variable = '变量'；  
let variable = '变量';   
// 不可变值：通常使用大写变量标识符
const CONSTANT = '常量'； 
```
* 良好的命名可以增加代码的可读性，减少不规范命名带来的低级错误
